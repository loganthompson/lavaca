Parameters:

  • pattern String

    The route URL pattern Route URL patterns should be in the form /path/{foo}/path/{*bar}. The path variables, along with query string parameters, will be passed to the controller action as a params object. In this case, when passed the URL /path/something/path/1/2/3?abc=def, the params object would be {foo: 'something', bar: [1, 2, 3], abc: 'def'}.

  • TController Function

    The type of controller that performs the action (Should derive from [[Lavaca.mvc.Controller]])

  • action String

    The name of the controller method to call

  • params Object

    Key-value pairs that will be merged into the params object that is passed to the controller action
